# Introducción

## Qué es Bocaraca Reservas

Bocaraca Reservas es un sistema web para academias de artes marciales orientado a la operación diaria de clases, alumnos y administración. Su objetivo es centralizar en una sola plataforma procesos que normalmente se manejan de forma manual, por chat o en hojas dispersas.

El proyecto está construido con un enfoque práctico:

- frontend estático en HTML, CSS y JavaScript vanilla
- Firebase como backend
- uso prioritario en celular y formato PWA
- administración simple sin depender de un servidor propio

## Qué problema resuelve

En una academia o gimnasio, varios procesos se vuelven repetitivos y propensos a errores:

- confirmar quién puede reservar
- controlar cupos por clase
- marcar asistencia
- saber quién está al día o vencido
- recibir comprobantes
- llevar reportes administrativos
- ajustar el calendario por feriados, seminarios o cambios especiales

Bocaraca Reservas agrupa todo esto dentro de una sola solución.

## Público objetivo

El sistema está pensado principalmente para:

- academias de artes marciales
- gimnasios con horarios y cupos
- administradores que necesitan control de alumnos
- profesores o personal que requieren acceso operativo
- alumnos que reservan y reportan pagos desde el celular

## Capacidades principales

### Para administración

- gestionar usuarios
- controlar mensualidades
- autorizar o bloquear acceso operativo
- configurar el calendario semanal
- crear excepciones por fecha
- revisar métricas
- generar reportes
- administrar roles

### Para clientes

- iniciar sesión
- ver el calendario de clases
- reservar espacios
- gestionar su perfil
- enviar comprobantes
- consultar disponibilidad

## Filosofía del proyecto

La aplicación prioriza:

- simplicidad operativa
- experiencia móvil
- despliegue barato
- mantenimiento incremental
- mejoras por módulo sin reescribir todo

No busca ser una plataforma genérica enorme, sino una solución concreta y útil para la operación real del negocio.

## Estado del proyecto

El proyecto ha evolucionado por módulos y mejoras sucesivas sobre una base funcional real. En su estado actual ya integra:

- autenticación
- panel administrativo
- panel cliente
- calendario de clases
- reservas
- asistencias
- mensualidades
- pagos y comprobantes
- reportes
- PWA

## Resumen

Bocaraca Reservas es una aplicación administrativa y operativa enfocada en academias, con fuerte prioridad en uso móvil, simplicidad técnica y control práctico del día a día.
